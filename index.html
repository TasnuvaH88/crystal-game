Crystal Guessing Game

Instructions

You will be given a randomized number.
There are four crystals below. Each crystal has a number assigned to it.
Every time you click on one, it will add that value to your total score.
The number of each crystal is hidden from you, so you need to figure out the right combination to match your score to the randomized number.

To win, you must end up with the same value as the number given to you!
If you go over the number, you lose. The values of each crystal change with each game!
